Scannerзұ»

a utilеҢ…йҮҢйқўпјҢи§Јжһҗеҹәжң¬зұ»еһӢе’Ңеӯ—з¬ҰдёІзұ»еһӢзҡ„ж–Үжң¬жү«жҸҸеҷЁ
b System.in зі»з»ҹиҫ“е…ҘжҢҮзҡ„жҳҜйҖҡиҝҮй”®зӣҳеҪ•е…Ҙж•°жҚ®пјӣжҺ§еҲ¶еҸ°иҫ“е…Ҙзҡ„ж•°жҚ®
c Scanner sc = new Scanner(System.in)пјӣи°ғз”Ёscзҡ„ж–№жі•иөӢеҖјз»ҷеҸҳйҮҸ

Scannerзұ»дёӯпјҡnextInt()гҖҒnext()гҖҒnextLine()еҢәеҲ«пјҡ

  • nextInt()пјҡеҸӘиҜ»еҸ–ж•ҙеҪўзҡ„ж•°жҚ®пјҢиҫ“е…ҘиҜ»еҸ–е®Ңд№ӢеҗҺпјҢе°Ҷе…үж Үж”ҫеңЁеҗҢдёҖиЎҢгҖӮжҚўеҸҘиҜқиҜҙпјҢеҪ“жҲ‘д»¬дҪҝз”ЁиҜҘж–№жі•ж—¶пјҢдјҡе°Ҷе…үж Үж”ҫеңЁиҜ»еҸ–ж•°еӯ—еҗҺйқўпјҢе№¶дё”жҳҜеҗҢдёҖиЎҢгҖӮ
  • nextLine()пјҡиҜ»еҸ–з©әж јпјҢзӣҙеҲ°д»ҘвҖҷ\nвҖҷдёәж Үеҝ—зҡ„иЎҢе°ҫгҖӮдёҖж—Ұиҫ“е…ҘиҜ»еҸ–е®ҢжҜ•пјҢиҜҘж–№жі•дјҡе°Ҷе…үж Үз§»еҲ°дёӢдёҖиЎҢејҖе§Ӣзҡ„дҪҚзҪ®гҖӮеҗёеҸ–еӯ—з¬ҰеүҚеҗҺзҡ„з©әж ј/Tabй”®пјҢеӣһиҪҰй”®жҲӘжӯў
  • next()пјҡиҜ»еҸ–иҫ“е…ҘеҶ…е®№зӣҙеҲ°йҒҮеҲ°з©әж јгҖӮе®ғдёҚиғҪиҜ»еҸ–дёӨдёӘеӯ—з¬Ұд№Ӣй—ҙзҡ„з©әж јгҖӮеҗҢж—¶е°Ҷе…үж Үж”ҫеңЁиҜ»еҸ–иҫ“е…ҘеҗҺйқўпјҢе№¶дё”еңЁеҗҢдёҖиЎҢгҖӮдёҚдјҡеҗёеҸ–еӯ—з¬ҰеүҚ/еҗҺзҡ„з©әж ј/Tabй”®пјҢеҸӘеҗёеҸ–еӯ—з¬ҰпјҢејҖе§ӢеҗёеҸ–еӯ—з¬ҰпјҲеӯ—з¬ҰеүҚеҗҺдёҚз®—пјүзӣҙеҲ°йҒҮеҲ°з©әж ј/Tabй”®/еӣһиҪҰжҲӘжӯўеҗёеҸ–

жіЁпјҡеӣ дёәеңЁиҝҷж ·зҡ„иҝҮзЁӢдёӯnextIntиҫ“е…Ҙз»“жқҹеҗҺдјҡдә§з”ҹдёҖдёӘеӣһиҪҰпјҢиў«nextlineиҜ»еҸ–пјҢеҸҜд»Ҙз”ЁnextпјҲпјүдјҡиҝҮж»ӨжҺүйӮЈдёӘеӣһиҪҰз¬ҰгҖӮ

Randomзұ»

a utilеҢ…йҮҢйқўпјҢз”ЁдәҺз”ҹжҲҗйҡҸжңәж•°
b public int nextInt(int n) иҝ”еӣһдёҖдёӘдјӘйҡҸжңәж•°пјҢиҢғеӣҙеңЁ 0 пјҲеҢ…жӢ¬пјүе’Ң жҢҮе®ҡеҖј n пјҲдёҚеҢ…жӢ¬пјүд№Ӣй—ҙзҡ„ int еҖјгҖӮ

дёӢйқўRandom()зҡ„дёӨз§Қжһ„йҖ ж–№жі•пјҡз”ҹжҲҗ[0,seed)еҢәй—ҙзҡ„ж•ҙж•°

гҖҖгҖҖRandom()пјҡеҲӣе»әдёҖдёӘж–°зҡ„йҡҸжңәж•°з”ҹжҲҗеҷЁгҖӮ

гҖҖгҖҖRandom(long seed)пјҡдҪҝз”ЁеҚ•дёӘ long з§ҚеӯҗеҲӣе»әдёҖдёӘж–°зҡ„йҡҸжңәж•°з”ҹжҲҗеҷЁгҖӮ
