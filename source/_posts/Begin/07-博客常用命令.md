# 常用博客语法

## 上传Git/Hexo语句：

```sh
//新建文件
hexo new "new file name"
//等于一次性执行了，清空、刷新、部署三个命令
hexo clean && hexo g && hexo d
//启动hexo
hexo s
-------------------------
git clone "仓库地址"

git add . 
//保存本地仓库的修改（commit）
git commit -m "messages"
//把本地仓库的修改，同步到github远程仓库上（push）
git push -u origin save
```

## Markdown 内嵌 HTML 使用

```html
//使用语法
<small>比默认字体小一号</small>
<big>比默认字体大一号</big>
<span style="background-color: #D3D3D3"><font size=4 color=#DC143C>默认字体，4号，使用十六进制表示颜色值</font></span>
//实现上标、实现下标
<sup>xxx</sup>
<sub>xxx</sub>
//关于图片处理的技巧
<img width=256 height=256 src="地址" />
<div align="center"><img src="地址" /></div>
<center>
<img width=256 height=256 src="地址" />
图 1 图片下标注
</center>
//高亮文本
<mark>高亮文本</mark>
```
