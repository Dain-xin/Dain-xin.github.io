## Hexo 常用命令

```bash
hexo new "FILE NAME"
//等于一次性执行了，清空、刷新、部署三个命令
hexo clean && hexo g && hexo d
```

## Git 常用命令

```bash
//1.查看本地仓库的状态
git status
//2.把本地文件夹的修改，同步到到本地仓库上（add）  add. 表示全部修改
git add . 
git add djx.txt
//3.保存本地仓库的修改（commit）
git commit -m "messages"
//4.把本地仓库的修改，同步到github远程仓库上（push）
git push
git push -u origin save
//5.清除cache未提交的重新提交
git rm --cache themes/next
//6.同步github内容到本地
git pull
//7.更新远程跟踪分支
git fetch origin
git fetch
git fetch origin master
//8.查看分支或切换
git checkout
git checkout save
//9.添加、删除、克隆
git clone https://github.com/Dain-xin/dain.git
git rm -r djx.txt
//10.合并两个内容
git pull -–rebase origin save
//11.删除分支
git branch -D save
git push origin -d save
//11.查看分支
git branch -a
git branch

///////////////////////////////////////////

1) 切换到基础分支，比如如master分支
git checkout master
2） 创建并切换到新分支
git checkout -b newBranch
3）更新分支代码并提交
git add *
git commit -m "init newBranch"
git push origin newBranch

///////////////////////////////////////////
//删除分支，仓库默认不是要删的分支，
git pull origin master --allow-unrelated-histories
git push origin -d save

///////////////////////////////////////////
echo "# dain" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Dain-xin/dain.git
//////////////////////////////////////////

```
