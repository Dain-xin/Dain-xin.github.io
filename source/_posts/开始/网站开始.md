## Hexo 常用命令

```bash
hexo new "FILE NAME"
//等于一次性执行了，清空、刷新、部署三个命令
hexo clean && hexo g && hexo d
```

## Git 常用命令

```bash
echo "# dain" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Dain-xin/dain.git
git push -u origin main


git clone https://github.com/Dain-xin/dain.git
git rm -r djx.txt
git add . 
git add djx.txt
git checkout
git fetch
git commit -m "delete all files"
git commit -m "dain"
git push
git pull

//1.查看本地仓库的状态
git status
//2.把本地文件夹的修改，同步到到本地仓库上（add）
// . 表示全部修改
git add . 
//3.保存本地仓库的修改（commit）
git commit -m "messages"
//4.把本地仓库的修改，同步到github远程仓库上（push）
git push
```
